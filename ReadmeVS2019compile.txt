Разработаны версии для AC22, AC23, AC24*, AC25 для платформы WIN
Компилированно и собрано в Microsoft Visual C++ 2019
Возможно использовать для младших версий:
AC 22 Microsoft Visual C++ 2015
AC 23 Microsoft Visual C++ 2017
AC 24 Microsoft Visual C++ 2017
AC 25 Microsoft Visual C++ 2019 (только VS2019)
* для просто АС24 произведена перенастройка с cMake на VS2019 (к сожалению оригинальный с cMake у меня не пошел на VS2019)

Номер целевой версии должен быть установлен в файле ResourceIds.hpp
// #define	AC_22 // для компиляции в АС22
// #define	AC_23_22 // для компиляции в АС23 и AC22
// Базовая версия 24 за комментировать остальные АС_2Х
// #define	AC_25 // для компиляции в АС25
