# Работа с тестовыми версиями аддона

Если есть желание и возможность присоединится к тестированию новых версий аддона, необходимо сделать следующее:

- Зарегистрироваться на GITHUB. Регистрация бесплатная, без регистрации не получится скачать промежуточные сборки.
- Зайти в раздел `Actions`
- Найти самую верхнюю строку с надписью `Add-On Build` и перейти в неё.
- Скачать нужный пакет и установить обычным способом.

# Изменённые функции для тестирования в версии 1.77 (на 10.07.2025)

**Новые функции**

- [Подсчёт ведомости отделки и построение элементов отделки](https://github.com/kuvbur/AddOn_SomeStuff/wiki/Finishing-ru)
- [Вывод объёмов компонент в выноску](https://github.com/kuvbur/AddOn_SomeStuff/wiki/Construction-Composition-ru)
- [Вывод информации МЕР элементов](https://github.com/kuvbur/AddOn_SomeStuff/wiki/Property-Commands-List-ru)
- [Построение 3д документов вдоль морфа-линии или ограждения](https://github.com/kuvbur/AddOn_SomeStuff/wiki/Build-Profile-Line-ru)
- Выравнивание чертежей на макетах по хотспотам
- Добавлено классическое округление в размеры (добавить в правило округления `ClassicRound `) #133
- Добавлен поиск по имени атрибута при записи в GDL параметр #134
- Добавлена повторная синхронизация при использовании в формулах обрабатываемых аддоном свойств #135
- Добавлено выделение связных дочерних элементов #138
- Добавлен вывод сообщения при вызове Показать связанные элементы (Если часть элементов находятся на другом этаже или не существует) #139
- Добавлен вывод версии аддона в отчёт, добавлен вывод сообщений об ошибках при создании элементов.
- Добавлена обработка замена двойных пробелов и образка пробелов в уникальных параметрах при создании спецификации (например, элементы, у которых в параметре только пробел будет считаться как без пробела. Двойные пробелы в расчётах будут заменены на одиночные)
- Добавлен вывод значения замены покрытия для объектов (в т.ч. окон и дверей) #137
- Добавил вторую версию функции Создать элементы (`Spec_rule_v2`)
- Суммирование текстовых значений при выводе функций Создать элементы теперь выводит не уникальные значения, а полные список значений.

**Исправлены ошибки**

- Связь с элементом может не срабатывать, если в проекте есть несколько свойств с описанием `Sync_GUID`, не видимых у связываемого элемента #132
- Исключена обработка отверстий при округлении размеров из-за бага в архикаде. Размеры, у которых хотя бы одна точка привязана к отверстию (Opening) обрабатываться не будет.
