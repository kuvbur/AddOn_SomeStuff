# Заполнение данных об изменениях по ГОСТ 21.1101

Функция совместима только с маркером изменений и штампом из библиотеки [kuvbur_Формат по ГОСТ](https://github.com/kuvbur/gdl_bibl/tree/master/kuvbur_%D0%A4%D0%BE%D1%80%D0%BC%D0%B0%D1%82%20%D0%BF%D0%BE%20%D0%93%D0%9E%D0%A1%D0%A2). Предназначена для заполнения штампов и маркеров изменений в соответствии с ГОСТ Р 21.101-2020.

Кнопка активна только при вызове с макета. Обработка производится всех видимых и зарезервированных маркерах изменения, расположенных на макетах. Для корректной работы необходимо также зарезервировать параметры макета.

При совместной работе необходимо соблюдать осторожность - **НЕ РЕЗЕРВИРОВАТЬ ЧУЖИЕ ЛИСТЫ**. **Отменить действия функции нельзя.**

Аддон проходит по каждому листу, собирая информацию о количестве и параметрах маркеров изменений. Для каждого листа с изменениями данные записываются в свойства макета `Somestuff_QtyIssue_1...n`, Примечание с номерами и типами изменений записываются в свойство `Somestuff_Note`. В каждый маркер изменения аддон записывает номер участка. Для регенерации листа (если это не произошло автоматически) достаточно перейти на другой лист.

После подгрузки свойств выделить все макеты и в свойства с именем `Somestuff_` записать пробелы.

Пример файла и схемы для подгрузки можно посмотреть [тут](https://github.com/kuvbur/gdl_bibl/tree/master/kuvbur_%D0%A4%D0%BE%D1%80%D0%BC%D0%B0%D1%82%20%D0%BF%D0%BE%20%D0%93%D0%9E%D0%A1%D0%A2)
