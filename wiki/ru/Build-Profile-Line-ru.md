## Построить профиль по линии
Создаёт 3д документы разрезов вдоль морфов-линий или ограждений. Используется для построения профилей сетей по ГОСТ.
1. Для каждого профиля создать рабочий лист с точкой (хотспотом) с номером пера 163. Этот рабочий лист будет использован для общего оформления
2. Построить морф или ограждение вдоль трассы сети. В ID морфа указать `ИМЯ_УЧАСТКА@МАСШТАБ`, например `В1.1@500`
3. Запустить функцию `Построить профиль по линии`. Выбрать морф-линию/ограждение. Будут созданы 3д документы по каждому участку. В 3д документах, по краям участка будут стоять точки (хотспоты). Их НЕ удалять.
4. Сохранить виды 3д документов, разместить на макете (можно не выравнивать, просто кучей)
5. На макете выбрать группу чертежей профиля и запустить функцию `Выровнять выделенные чертежи`.

При изменении положения трассы на плане передвинуть морф/ограждение, повторить пункты выше. Уже созданные 3д документы будут обновлены, новые добавлены.
