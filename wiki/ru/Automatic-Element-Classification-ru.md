## Назначение классификации по свойству

Аддон позволяет автоматически назначать и синхронизировать классификацию элементов на основе значений свойств.

### Включение классификации для неклассифицированных элементов

Автоматическое присвоение класса элементам без классификации

Добавьте в описание класса значение: `some_stuff_class`  
_Пример настройки:_  
![изображение](https://github.com/user-attachments/assets/b87efbe3-094c-4f9a-895d-57a7bf38cc2b)

### Смена класса на основании значения свойства

Динамическое изменение класса элемента при изменении связанного свойства

1. Создайте текстовое свойство
2. В описании свойства укажите:  
   `Sync_to{Class:ИМЯ_КЛАССИФИКАЦИИ}`  
   _(Поиск класса производится по его ID)_
3. Добавьте в свойство свою формулу, которая будет давать в результате ID класса

_Пример настройки:_  
![изображение](https://github.com/user-attachments/assets/2e1c08cb-1b36-4a4a-8cae-1f6a6da39f69)

### Вывод информации о классе

Создайте свойство с описанием:  
`Sync_from{Class:ИМЯ_КЛАССИФИКАЦИИ; FullName}`

- Выводит полное имя класса элемента

## Отключение обработки классификации

**Варианты:**

- Использование основного флага обработки:  
  Свойство `Sync_flag` (полное отключение обработки элемента)
- Создание специализированного флага:  
  Свойство с описанием `Sync_class_flag` (отключает только обработку классификации)
