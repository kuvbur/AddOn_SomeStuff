# WIN
1. В разделе [Releases](https://github.com/kuvbur/AddOn_SomeStuff/releases) качаем последнюю версию дополнения нужной версии с суффиксом WIN
2. Если Архикад запущен - закрываем.
3. Копируем скачанный файл в папку с расширениями Архикада (например, *C:\Program Files\GRAPHISOFT\ARCHICAD 24\Расширения ARCHICAD\Дополнения*)
4. Запускаем архикад.

# MAC
1. В разделе [Releases](https://github.com/kuvbur/AddOn_SomeStuff/releases) качаем последнюю версию дополнения нужной версии с суффиксом MAC
2. После скачивания и распаковки архива :
   - Откройте Терминал
   - Перейдите в папку с аддоном:
   - Снимите атрибуты карантина:
     ```bash
     xattr -cr SomeStuff.bundle
     ```
     *(При необходимости используйте `sudo xattr -cr SomeStuff.bundle`)*
   - **Подпишите аддон** (обязательный шаг для новых версий macOS):
     ```bash
     codesign --force --deep --sign - SomeStuff.bundle
     ```
4. Запускаем архикад.
5. В менеджере дополнений нажимаем Добавить и указываем на аддон.

# Добавление команд аддона в меню / табло команд

1. Открыть меню Параметры->Окружающая среда
2. Зайти в пункт Меню или Табло команд
3. В правой части окна создать новое меню или табло команд либо открыть существующее
4. В левой части выбрать Все команды по теме, перейти в список Расширения->SomeStuff
5. Добавить нужные команды
