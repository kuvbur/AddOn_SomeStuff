# Округление значений

В качестве строки с форматом округления аддон воспринимает часть имени свойства после точки. Эта часть должна содержать букву m.

Например, для строки-шаблона состава конструкции `%толщина.2mm%`, но не `%толщина.2%`

Для синхронизации параметров запись будет иметь вид, например Sync_from{A.1mm}

Возможные варианты (примеры):

- `%толщина.1mm%` - вывод в миллиметрах, с одним знаком после запятой и подавлением нулей в конце, например 12 или 12,1
- `%толщина.01mm%` - вывод в миллиметрах, с одним знаком после запятой, например 12,0 или 12,1
- `%толщина.01mp%` - вывод в миллиметрах, с одним знаком после запятой, с заменой разделителя дробной части на точку.
